# pc-draftkings-archive

## 项目简介

`pc-draftkings-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，仅作为静态页面的存档与展示集合，方便版本管理和历史记录查阅。

## 目录说明

```
pc-draftkings-archive/
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的静态资源（如图片、样式表、脚本）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个独立 HTML 页面以文件或子目录形式存放，命名清晰，便于识别。
- `assets/`：资源文件按类型或页面归类，保持路径整洁。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，内容可能包含不同时期、不同主题的静态页面。每个页面都保留了原始结构与样式，未做额外修改或优化。

- 页面文件可直接在浏览器中打开浏览。
- 部分页面可能引用外部资源，建议在联网环境下查看。
- 归档文件按时间或主题命名，便于检索。

## 维护说明

- 本仓库仅用于归档与展示，不提供实时更新或动态内容。
- 如需新增归档页面，请将 HTML 文件放入 `pages/` 目录，并确保引用资源路径正确。
- 不建议对已有归档页面进行内容修改，以保持历史记录的完整性。
- 欢迎提交 Issue 或 Pull Request 以协助整理或修复链接问题。
